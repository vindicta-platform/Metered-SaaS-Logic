# Metered SaaS Logic

**Usage-based billing made simple.**

Track usage, calculate costs, and manage billing cycles for metered SaaS applications.

## Features

- **Event Tracking** — Record usage events
- **Aggregation** — Hourly/daily summaries
- **Credit Tracking** — Pre-paid balance management

## Installation

```bash
uv pip install git+https://github.com/vindicta-platform/Metered-SaaS-Logic.git
```
